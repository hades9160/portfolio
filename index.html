MAMA'S RECIPE is a personalized and customizable meal planner with dietitian advisory features. This full-stack application combines mobile and web development to deliver a culturally relevant meal planning solution specifically designed for Filipino users.

The application bridges the gap between users seeking personalized nutrition guidance and licensed dietitians by providing a subscription-based platform that offers expert-approved meal plans, real-time notifications, and direct communication channels.

üéØ Key Features

  • Personalized meal plan recommendations based on health goals
  • Direct messaging between users and dietitians
  • Subscription-based access to dietitian meal plans
  • Meal calendar integration with time reminders
  • Offline access to downloaded meal plans
  • QR-based payment system for subscriptions
  • Descriptive and prescriptive analytics for personalization
  • Admin dashboard for user and dietitian management
  • Real-time notifications for new meal plans
